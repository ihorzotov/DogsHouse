Домик для маленьких собак

Оформление индивидуальной зоны для собаки — проявление заботы и ласки. Даже собаке, которая живет в общем доме личный дом будет шикарным презентом, элементом послушания и воспитания.

Домик подойдет для таких пород:

  • Йоркширский терьер, той терьер, джек рассел терьер
  • Мопс, французский бульдог
  • Шпиц, пти брабансон, карликовый пинчер
  • Другие маленькие породы
